
		oldsce.x v1.10 SC61860 CPU エミュレータ

								小笠原博之

  遙か昔 (1994年の2月らしい) に作った CPU エミュレータです。CPU のみを完璧に
エミュレートします。SC61860 とは、かつて SHARP のポケコンで使われていた CPU
です。代表的な機種に PC-1250/51/55/45, PC-1260/61/62/80, PC-1350/60/60K,
PC-1401/02～75 などがあります。また、SHARP の電子手帳 PA-7000/8500/S1/X1 シ
リーズなど、2/4行表示で IC カードが使えるタイプのものにも使われていました。

  この CPU は命令体系が独自で、他の CPU に一切似ていません。さらに SHARP は
このマシン語命令の資料を公開していませんでした。にもかかわらず、当時のパワー
ユーザーは自力で命令を解析し、推測して、それらの命令セットを調べ上げてしまっ
たのです。それら解説記事が当時の雑誌(たしか初期の TheBASIC、B-NUMBER てのに
も載っていた)に掲載されたときは、物凄い驚きでした。

  その後、工学社系の雑誌や書籍で、SHARP の海外向け資料を元にしたマシン語解
説本などが登場しましたが、やはり当時のユーザーにとって極めて思い入れの強いマ
シンと CPU であったことは間違いありません。

  このエミュレーターが、工学社系の解説書には一切載っていない、CPU の裏技とも
言うべき CPU の裏の動作まできちんと対応しているのは、当時1つ1つ手で調べなが
ら解析した人がいたからにほかなりません。

  このエミュレータは CPU 命令のみの動作を行い、I/O などは一切対応していませ
ん。純粋な CPU 命令を動かすのみになっています。もちろん、CPU だけでなくハー
ドウエアエミュレーションを追加すればポケコンエミュレータになるわけですが、
私はそこまでは行いません。当時の懐かしいあの感動が忘れられないという方は、
ぜひ拡張なり改造なり挑戦してみてください。


  このプログラムは X68K 版をそのまま Windows95(32bit) 上でコンパイルしたもの
です。Windows95 の DOSプロンプト上で動作します。(NTは未確認ですが多分動作す
るのでは)
